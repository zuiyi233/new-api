## 0. 证据标记与精度声明

### 0.1 证据来源标记
- **[图像可见]**：可从参考图直接观察到（结构、相对位置、是否存在某元素）。
- **[代码实值]**：当前实现代码中可直接读取的明确值（CSS/JS 常量）。
- **[近似推断]**：图片无法给出绝对像素或完整状态时，基于图像与实现的合理近似。

### 0.2 精度声明（避免伪精确）
- 由于会话中的参考图未提供原始测量标尺与设计稿标注，**所有“绝对像素级”结论仅在代码值层面可精确**；
